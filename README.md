# Event Discovery & Tracking Tool

## Overview
Automated tool to fetch city-wise events, store them in Google Sheets,
and keep data updated with scheduled runs.

## Features
- City-based event fetching
- Google Sheets integration
- Event deduplication using hashing
- Expired event handling
- Cron-based automation

## Tech Stack
- Node.js
- Google Sheets API
- Cron

## How to Run
1. npm install
2. Add Google service account credentials
3. node index.js
